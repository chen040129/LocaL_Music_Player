# 音乐数据记忆功能

## 功能说明

应用现在具有自动记忆功能，可以保存您导入的音乐数据，下次打开应用时无需重新导入。

## 主要特性

1. **自动保存**
   - 扫描音乐后自动保存到本地
   - 移除文件夹后自动更新本地数据
   - 清空音乐时自动清除本地数据

2. **自动加载**
   - 应用启动时自动加载上次保存的音乐数据
   - 自动恢复已扫描的文件夹列表

3. **数据存储位置**
   - Windows/Linux/MacOS: 应用支持目录
   - 移动设备: 应用文档目录

## 使用方法

无需任何额外操作，功能自动生效：
1. 首次使用时，扫描音乐后数据会自动保存
2. 下次打开应用时，数据会自动加载
3. 如果需要清除所有数据，可以使用"清空所有音乐"功能

## 技术实现

- 使用 `StorageService` 管理本地数据存储
- 使用 JSON 格式保存音乐信息和文件夹列表
- 通过 `MusicProvider` 统一管理数据的加载和保存
- 在 `HomeScreen` 初始化时自动加载数据

## 注意事项

- 音乐封面图片不会保存到本地（节省存储空间）
- 如果音乐文件被移动或删除，可能需要重新扫描
- 清除应用数据会丢失所有保存的音乐信息
