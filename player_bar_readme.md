
# 可拖动音乐栏功能说明

## 功能概述

可拖动音乐栏功能允许底部音乐控制栏在主窗口内自由拖动，类似于其他音乐软件的桌面歌词功能。音乐栏可以：

1. 在主窗口内自由拖动
2. 不受原有布局限制
3. 保存拖动位置，下次启动时恢复
4. 保持原有的音乐控制功能

## 使用方法

1. 启动主应用后，音乐栏会显示在窗口底部

2. 点击并拖动音乐栏可以将其移动到窗口内的任意位置

3. 松开鼠标后，音乐栏会停留在当前位置

4. 窗口位置会自动保存，下次启动时恢复到上次的位置

## 技术实现

### 主要组件

1. **DraggablePlayerControlBar** - 可拖动的音乐控制栏组件
   - 支持自由拖动
   - 支持位置保存和恢复
   - 保留原有的音乐控制功能

### 配置选项

在 `player_control_bar_draggable.dart` 中可以配置以下选项：

- 初始位置：通过 `_position` 变量设置
- 位置保存：使用 SharedPreferences 保存位置信息

## 注意事项

1. 音乐栏在主窗口内拖动，不会超出窗口边界
2. 窗口位置保存在 SharedPreferences 中，清除应用数据会丢失位置信息
3. 音乐栏保留原有的所有播放控制功能
4. 首次启动时，音乐栏会显示在窗口底部

## 常见问题

**Q: 音乐栏可以拖动吗？**

A: 是的，音乐栏可以在主窗口内自由拖动到任意位置，拖动结束后位置会自动保存。

**Q: 主窗口最小化时，音乐栏会消失吗？**

A: 会，音乐栏在主窗口内，会随主窗口一起最小化。

**Q: 如何重置音乐栏位置？**

A: 可以通过 SharedPreferences 清除保存的位置数据，音乐栏会恢复到默认的底部位置。

**Q: 音乐栏会超出窗口边界吗？**

A: 不会，音乐栏在主窗口内拖动，不会超出窗口边界。
